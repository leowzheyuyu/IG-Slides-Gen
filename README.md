# IG-Slides-Gen

  This document is a comprehensive **instructional framework** for an AI-driven **Instagram Carousel Generator**.[cite: 1] It transforms a simple text prompt into a professionally designed, multi-slide carousel using web technologies (HTML/CSS) specifically optimized for social media marketing.[cite: 1]

### **What is this?**
It is a **design system and export protocol** that allows an AI to act as a graphic designer and developer.[cite: 1] Instead of just giving you text, the AI uses these instructions to:
*   **Generate a Brand Identity:** Derives a full 6-token color palette and typography system from a single primary color or brand name.[cite: 1]
*   **Build a Swipeable Preview:** Creates a self-contained HTML "Instagram Frame" (420px wide) where you can interactively swipe through the slides.[cite: 1]
*   **Automate Professional Export:** Uses a Python script with Playwright to capture high-resolution (1080x1350px) PNGs of each slide, ensuring they are perfectly formatted for Instagram's 4:5 aspect ratio.[cite: 1]

---

### **How to use it**
To use this system, you (the user) follow a structured workflow with the AI:

#### **1. Initiation & Branding**
You provide the core topic (e.g., "Make a carousel about AI Automation").[cite: 1] The system will then ask you for:
*   **Brand Name & Handle:** For the header and footer.[cite: 1]
*   **Primary Color:** To generate the accents and backgrounds.[cite: 1]
*   **Style/Tone:** Choosing between "Editorial," "Modern," "Technical," etc.[cite: 1]

#### **2. Content Generation**
The AI follows a "Standard Slide Sequence" to ensure high engagement:[cite: 1]
*   **The Hook (Slide 1):** A bold hero statement on a light background.[cite: 1]
*   **The Problem/Solution (Slides 2-3):** Identifying a pain point and introducing the answer.[cite: 1]
*   **The Value (Slides 4-6):** Feature lists, "how-to" steps, or technical details using pre-built components like "Strikethrough pills" or "Numbered steps."[cite: 1]
*   **The CTA (Slide 7):** A final call to action on a brand-gradient background.[cite: 1]

#### **3. Review & Refinement**
The AI generates an HTML preview. You can swipe through it to check:
*   **Visual Rhythm:** Alternating light and dark slides.[cite: 1]
*   **UI Elements:** Ensuring the progress bar and swipe arrows are correctly placed.[cite: 1]
*   **Consistency:** Checking that fonts and spacing are uniform.[cite: 1]

#### **4. High-Resolution Export**
Once you approve the design, the AI uses the provided **Python/Playwright script**.[cite: 1] This script:
1.  Loads the HTML at a specific scale factor ($2.5714$).[cite: 1]
2.  Snaps screenshots of each individual slide.[cite: 1]
3.  Saves them as **1080x1350px PNGs**, ready for you to upload directly to Instagram.[cite: 1]

# Where do I use it

You can use this system across three specific environments to move from a text idea to a final Instagram post:

### **1. Within an AI Chat Interface**
This is where the "Design System" lives.[cite: 1] You provide these instructions to an AI (like me) to:
*   **Process Brand Details:** Input your brand name, handle, and primary colors to generate the custom palette.[cite: 1]
*   **Preview the Carousel:** The AI generates a self-contained HTML file that provides a "swipeable" preview of your slides directly in the chat window.[cite: 1]
*   **Iterate on Content:** You can ask for specific changes to the text or layout of individual slides before they are finalized.[cite: 1]

### **2. In a Python Development Environment**
To turn the HTML preview into actual images, you use the provided **Export Script** on your computer.[cite: 1]
*   **Local Setup:** You will need **Python** installed along with the **Playwright** library.[cite: 1]
*   **The Workflow:** You save the HTML code generated by the AI to a file, then run the Python script to automate "taking pictures" of each slide.[cite: 1]
*   **Cloud Alternatives:** If you don't want to set it up locally, you can use environments like **Google Colab** or **Replit** to run the Python export.[cite: 1]

### **3. On Instagram**
The ultimate destination for the output of this project is your **Instagram feed**.[cite: 1]
*   **The Format:** The script generates **1080×1350px PNG images**, which is the standard 4:5 aspect ratio for high-quality Instagram carousels.[cite: 1]
*   **Posting:** You simply transfer these PNGs to your phone and upload them as a multi-image post.[cite: 1]
